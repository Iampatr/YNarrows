# Стрелки, сигналы (YNarrows)

HUD-приложение для второго дисплея Geely Monjaro. Отображает навигационные подсказки (стрелки поворотов, расстояние, улицу), сигналы светофоров и дорожные события (камеры, ДТП, ремонт) с Yandex Navi и 2GIS.

## Как работает

- **NotificationListenerService** — перехватывает уведомления Yandex Navi / 2GIS
- **RemoteViewsUtils** — извлекает содержимое RemoteViews (иконки маневров, текст, светофоры) через рефлексию
- **HUDActivity** — отображает полученные данные на втором дисплее (displayId=2) в автомобиле

## Возможности

- Стрелки поворота (налево, направо, разворот, кольцо и т.д.)
- Расстояние до манёвра
- Название улицы
- Сигналы светофора (зелёный, жёлтый, красный + секунды)
- Дорожные события: камеры, ДТП, ремонтные работы
- Отображение уведомлений от 2GIS (музыка/название трека)

## Требования

- Android 9+ (minSdk 28)
- Установка как system/priv-app (требуются системные разрешения: `MANAGE_ACTIVITY_STACKS`, `DISABLE_HIDDEN_API_CHECKS`, `WRITE_SECURE_SETTINGS` и др.)
- Включённый Notification Listener Service для YNarrows
- Включённый Accessibility Service для YNarrows
- Yandex Navi или 2GIS установлены и запущены

## Сборка

```bash
./gradlew assembleDebug
```

JDK 17, Gradle 8.11.1, AGP 8.10.1.
