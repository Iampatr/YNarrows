# YNarrows (Стрелки, сигналы)

HUD-приложение для второго дисплея Geely Monjaro. Отображает навигацию с Yandex Navi и 2ГИС: стрелки поворотов, расстояние до манёвра, название улицы, сигналы светофора и дорожные события (камеры, ДТП, ремонт).

## Как это работает

Два независимых источника данных:

1. **NotificationListenerService** — перехватывает уведомления навигационных приложений, через рефлексию RemoteViews извлекает иконки маневров, текст, сигналы светофоров
2. **AccessibilityService** (`NodeInfoForNavi`) — читает accessibility-дерево Yandex Navi по viewID для получения манёвра, расстояния и улицы

Оба пути отправляют broadcast-и в `HUDActivity`, которая отображает данные на втором дисплее (displayId=2, bounds 0,721,1920,1080) в режиме свободного окна (windowingMode=5).

## Возможности

- Стрелки поворота (25+ типов: налево, направо, разворот, кольцо с номерами съездов и т.д.)
- Расстояние до манёвра и название улицы
- Сигналы светофора (зелёный, жёлтый, красный + секунды до смены)
- Дорожные события: камера, ДТП, дорожные работы
- Уведомления 2ГИС (отображение текущего трека)

## Требования

- Android 9+ (minSdk 28, targetSdk 35)
- Установка как system/priv-app (требуются системные разрешения)
- Включённый Notification Listener Service
- Включённый Accessibility Service
- Yandex Navi или 2ГИС
