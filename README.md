# YNarrows (Стрелки, сигналы)

HUD-приложение для второго дисплея автомболией Lynk Co (Geely Monjaro platform) 01/03/05/09 с операционной системой ГУ - Андроид 9 (oneOS). Отображает навигацию с Yandex Navi и 2ГИС: стрелки поворотов, расстояние до манёвра, название улицы, сигналы светофора и дорожные события (камеры, ДТП, ремонт).

## Как это работает

Два независимых источника данных:

1. **NotificationListenerService** — перехватывает уведомления навигационных приложений, через рефлексию RemoteViews извлекает текстовые представления VIEW приложения в уведомлении: сигналы светофоров, камеры, маневры и иные события. По рефлексии предопределенных пользовательских представлений дествий в RemoteViews определяет устанавливаемые значения и на основе полученных данных интерпретирует свершившееся событие с последующим выводом информации на ХУД.
    
2. **AccessibilityService** (`NodeInfoForNavi`) — читает accessibility - события в нодах Yandex Navi по заранее известным значениям. Для их предопределения предусмотрен также режим отладки - чтение дерева нодов текущего отображения основного дисплея до 9 уровня вниз - для нагладной отладки на ходу.

Оба пути отправляют broadcast-и в `HUDActivity`, которая отображает данные на втором дисплее (displayId=2, bounds 0,721,1920,1080) в режиме свободного окна (windowingMode=5) и прозрачной темой окна, чтобы не перекрывать собой другие данные на HUD.

3. Предоставляет подробную информацию о всех маневрах, предусмотренных оригинальным приложением, включая номера съездов с кругового движения и номера поворотов (для AccessibilityService - foreground mode в Яндекс навигаторе), а также дополнително к маневрам - камеры и сигналы светофоров с отсчетом обратного времени (в режиме NotificationListenerService - background mode в Яндекс навигаторе). Маневры для 2gis обеспечиваются лишь в режиме работы background mode для 2gis. Для 2gis маневр определяется прямой передачей иконки уведомления на ХУД. Расстояние и улица - из описания и заголовка уведомления.


## Возможности

- Стрелки поворота (25+ типов: налево, направо, разворот, кольцо с номерами съездов и т.д.)
- Расстояние до манёвра и название улицы
- Сигналы светофора (зелёный, жёлтый, красный + секунды до смены)
- Дорожные события: камера, ДТП, дорожные работы
- Уведомления 2ГИС (отображение текущего трека)

## Требования

- Android 9+ (minSdk 28, targetSdk 35)
- Установка как system/priv-app (требуются системные разрешения)  - возможна также и работа в качетсве пользовательского приложения.
- Включённый Notification Listener Service
- Включённый Accessibility Service
- Yandex Navi или 2ГИС

## Сборка

```
./gradlew assembleDebug
```
